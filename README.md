# 🎰 로또 번호 추천기

## 개요
로또 번호 추천기는 과거 20회차의 로또 번호를 분석하여 10가지 다른 방법으로 로또 번호를 추천해주는 웹 기반 애플리케이션입니다.

## 기능

### 1. 데이터 분석
- **자주 나온 번호**: 과거 20회차에서 가장 자주 나온 상위 10개 번호 표시
- **미사용 번호**: 한 번도 나오지 않은 번호들 표시

### 2. 10가지 추천 방법
1. **최고 인기 번호들**: 가장 자주 나온 상위 6개 번호
2. **중간 인기 번호들**: 중간 정도 자주 나온 번호들
3. **미사용 번호 포함**: 나오지 않은 번호를 포함한 추천
4. **홀수/짝수 균형**: 홀수 3개, 짝수 3개로 균형잡은 선택
5. **번호 범위 균형**: 1~22번: 3개, 23~45번: 3개로 분산
6. **최근 인기 번호들**: 최근 7회차에서 자주 나온 번호
7. **과거 인기 번호들**: 처음 7회차에서 자주 나온 번호
8. **분산된 번호**: 1~45 범위에 고르게 분산된 번호
9. **클러스터 번호**: 비슷한 범위의 번호들로 구성
10. **행운의 선택**: 무작위 선택 (운에 맡기기)

## 데이터 소스

- **백업**: `lotto-data.json` - 로컬 저장된 샘플 데이터

## 기술 스택
- **HTML5**: 페이지 구조
- **CSS3**: 반응형 디자인
- **JavaScript (Vanilla)**: 로또 분석 및 추천 로직
- **Fetch API**: 외부 데이터 로드

## 주요 클래스: `LottoAnalyzer`

### 메서드
- `fetchLottoData()`: 외부 API 또는 로컬 데이터에서 로또 번호 가져오기
- `calculateFrequency()`: 각 번호의 빈도수 계산
- `getAnalysisResult()`: 분석 결과 반환
- `generateRecommendations()`: 10가지 추천 번호 생성

## 사용 방법

1. 페이지 로드 후 "로또 번호 분석하기" 버튼 클릭
2. 로또 번호가 로드되고 분석됩니다
3. 자주 나온 번호와 나오지 않은 번호를 확인합니다
4. 10가지 추천 방법 중 원하는 번호를 선택합니다

## 업데이트 방법
`lotto-data.json` 파일을 새로운 로또 번호로 업데이트하면 자동으로 분석과 추천이 변경됩니다.

### lotto-data.json 형식
```json
[
  [7, 14, 18, 23, 31, 42],
  [11, 16, 25, 32, 38, 44],
  ...
]
```
각 배열은 한 회차의 6개 번호입니다. 20개 배열이 필요합니다.
